# Опис:
## WiFi-UPS(ДБЖ для роутера) - це пристрій який забезпечить безперебійне живлення для вашого WiFi роутера та ONU терміналу. Він складається з доступних компонентів та електронних модулів, що робить його простим для повторення. У цьому репозиторії описано будову та принцип роботи, список необхідний компонентів, інструкція зі збірки.

# Технічна інформація:
Особливістю цього безперебійника є безшовне перемикання на резервне живлення, тобто роутер не буде перезавантажуватись під час відключень світла. Живити безперебійник можна блоком від роутера(9V/12V 600мА-2А) або стандартним блоком живлення від смартфона по USB-typeC(5V 1-3A). Присутні 2 входи та 2 виходи.Характеристика вхідних портів: IN1: джек 5.5x2.1мм напргуа від 6 до 20 Вольт, струм до 3 Ампер; IN2: гніздо typeC напргуа 4.5 - 5.5 вольт, струм до 3 Ампер. Характеристика вихідних портів: OUT1,OUT2: джек 5.5x2.1мм напруга 9/12 вольт(при потребі можна виставити іншу в діапазоні 5-25 Вольт), Струм до 1 Амперу на порт. Максимальна вихідна потужність на 2 порти складає 24W. Також на передный панелы розташований ындикатор заряду акумулятора та вимикач. У якості елементів живлення використовуються 4 Li-Ion акумулятори формату 18650 та ємністю >1500мА на елемент.
